# Rock-Paper-Scissors
This is a Console-only Rock, Paper, Scissors Game created using JavaScript and HTML. 

This was created for The Odin Project course's JavaScript foundations lessons. 

# Purpose

This was made so I could sharpen the following skills:

- **JavaScript Logic Flow:** Handling player input and randomized computer choices.
- **Git:** Ability to use Git and GitHub effectively.
- **Developer Tools:** Navigating browser (Google Chrome) developer tools.
- **Debugging:** Using the tools to perform debugging using different types of breakpoints such as line-of-code, conditional, and logpoint.
- **Execution Context:** Using the tools to understand the concept of the Call Stack and JavaScript's single-threaded nature.

# How to Play

1. Clone this repository.
2. Open index.html in your browser.
3. Open the Developer Tools (right-click and choose 'Inspect' on any browser).
4. Find the 'Console' (or similar name) tab. 
5. Call the game function by typing:
```js
playGame();
```
6. Enjoy!
